Using the count() function

As we discussed in class, the count() function can be a useful way to summarize data for plotting. Here’s an example of how this works:

mpg %>% count(year)

Tell us a story

As we mentioned in one class, one important principle of creating data visualizations is using them to tell a clear story about some aspect of the data. We would like you to tell us something interesting about the data. By interesting we don’t mean it has to be anything earth-shattering, but just a coherent conclusion about the data that is communicated by your plot. Make it look nice insofar as it serves this end, but there’s no need to go overboard. For more about what we mean by tell a story, we will refer you to this book chapter which shows some examples of storytelling with the mpg dataset: https://r4ds.had.co.nz/graphics-for-communication.html Of course we would like your story to be different from the examples in that chapter. Be creative and have fun with it!
